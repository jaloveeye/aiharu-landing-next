# Fragment Memory Skills Index

이 폴더는 **세션이 끊겨도 맥락을 이어가기 위한 “파편(≤300자) 기반 기억 운용 스킬”** 모음입니다.

## 기본 규율(항상)
- 작업 시작 전: `fm_10_preflight-review.md` (버그/확장/컨벤션/보안/모호함 점검 후 요약 보고)
- 코드 변경 시: Git Flow 규칙을 따른다 → `fm_15_git-flow.md`
- 세션 시작: `context()`
- 작업 중: 중요한 사실/결정/에러/절차/관계는 `remember()`로 즉시 파편화
- 과거 경험 필요: `recall()`
- 세션 종료(또는 큰 마일스톤): `reflect()`
